# results ui 에이전트 규칙

- 웨이브 클리어, 스테이지 클리어와 스테이지 실패를 구분한다.
- 시스템 실패 용어는 `스테이지 실패`, 재시작 행동은 `리트라이`를 사용한다.
- 스코어, 콤보, 코인, 주요 유물 발동과 개선 가능한 실패 단서를 표시한다.
- 다음 행동을 다음 웨이브, 다음 스테이지, 메인 로비 또는 리트라이로 정확히 표현한다.
- 진행 상태를 직접 변경하지 않고 하나의 사용자 의도를 전달하며 처리 중 재입력을 막는다.

# gameplay ui 에이전트 규칙

- 현재 스코어, 목표 스코어와 남은 목표를 계속 표시한다.
- 코인은 스코어와 다른 아이콘, 색과 위치 계약을 사용한다.
- 콤보, 배수, 선택된 플리퍼와 공 상태를 원인과 함께 즉시 갱신한다.
- HUD와 숫자 팝업이 공, 예상 경로와 보스 위험 범위를 가리지 않게 한다.
- 공 라이프 표시는 `D-01`, `D-02`가 결정되기 전 임시 규칙으로 확정하지 않는다.
- 목표 도달, 코인 획득과 콤보 경계의 표시 지연·누락·중복을 검증한다.

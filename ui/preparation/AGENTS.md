# preparation ui 에이전트 규칙

- 보유 유물, 보드 슬롯, 현재 배치와 검증 실패 이유를 공개 데이터로 표시한다.
- 드래그·클릭 입력은 배치 명령으로 전달하고 UI가 배치 상태를 임의 확정하지 않는다.
- 유물 확인과 배치 단계를 생략하지 않되 정확한 진입 시점은 `D-06` 결정을 따른다.
- 배치 목적과 위치 차이를 읽을 수 있게 안전·고득점·보스 공격 경로 정보를 가리지 않는다.
- 준비 완료 연타와 검증 중 입력을 차단하고 최종 승인된 배치를 다시 표시한다.

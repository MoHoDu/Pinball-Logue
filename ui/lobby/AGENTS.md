# lobby ui 에이전트 규칙

- 공식 명칭은 `메인 로비`이며 월드맵 또는 스테이지 선택 화면과 혼용하지 않는다.
- 스테이지 1·2·3의 현재, 완료, 진입 가능과 잠김 상태를 공개 진행 데이터로 표시한다.
- 진입 가능 여부를 계산하거나 직접 씬을 열지 않고 선택 의도만 전달한다.
- 잠긴 스테이지 선택, 연타와 로딩 중 재입력을 차단하고 이유를 표시한다.
- 키보드 포커스 순서와 현재 선택 상태를 시각적으로 구분한다.

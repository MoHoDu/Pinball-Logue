# capture tools 에이전트 규칙

- 증거마다 작업 ID, 날짜, 환경, 빌드, 입력, 시드, 실행 횟수와 측정 단위를 기록한다.
- 캡처와 녹화는 실제 검증 장면과 핵심 UI가 보이도록 하고 민감 정보와 다른 앱을 포함하지 않는다.
- 큰 영상과 바이너리는 저장 위치를 사용자와 합의하기 전 Git에 추가하지 않는다.
- 측정하지 않은 값을 추정하지 않고 `미측정`으로 남긴다.
- 캡처 실패가 테스트 결과를 성공으로 바꾸지 않으며 로그와 원본 결과를 보존한다.

# reward economy 에이전트 규칙

- 검증된 코인 획득과 구매 차감을 거래 식별자와 함께 원장 형태로 기록한다.
- 스코어를 코인으로 자동 변환하거나 같은 값으로 취급하지 않는다.
- 중복 거래, 음수 획득·차감과 잔액 초과 차감을 거부한다.
- 코인 유지 범위는 `D-04`, 리트라이 초기화는 `D-03` 결정 전까지 외부 정책으로 받는다.
- 평균 획득량, 중앙값, 범위, 안정 보상 가격과 2개 구매 가능 비율을 같은 플레이 표본으로 검증한다.

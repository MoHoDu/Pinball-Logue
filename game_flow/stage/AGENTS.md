# stage 진행 에이전트 규칙

## 책임

- 한 스테이지의 준비, 일반 웨이브, 보상, 보스, 클리어와 실패 상태를 관리한다.
- 일반 웨이브 1·2·3 각각의 클리어 뒤 보상 구매 단계를 한 번씩 연다.
- 세 번째 보상 완료 뒤 보스 웨이브로 진입한다.
- 실패 뒤 같은 스테이지의 일반 웨이브 1부터 시작하는 리트라이 결과를 제공한다.

목표 스코어 계산, 보상 카드와 가격, 보스 전투, 화면 전환은 소유하지 않는다. 실패 시 초기화할 재화·보상·배치의 범위는 `D-03` 결정 전까지 확정하지 않는다.

## 전환 규칙

```text
준비
→ 일반 웨이브 1 → 보상
→ 일반 웨이브 2 → 보상
→ 일반 웨이브 3 → 보상
→ 보스
→ 클리어
```

- 일반 웨이브 클리어 전에는 해당 보상 단계에 진입할 수 없다.
- 보상 완료 결과 하나는 다음 웨이브 또는 보스로 한 번만 진행시킨다.
- 보스 클리어만 스테이지 클리어를 만들 수 있다.
- 어느 웨이브에서든 유효한 실패 결과가 오면 스테이지 실패로 전환한다.
- 리트라이 승인을 받은 실패 상태만 일반 웨이브 1의 준비 상태로 돌아간다.

## 필수 검증

- 일반 웨이브 3회와 보상 3회의 순서와 횟수가 정확한지 확인한다.
- 세 번째 보상 전 보스 진입과 보스 전 스테이지 클리어가 불가능한지 확인한다.
- 완료·실패 결과 중복으로 상태가 두 번 바뀌지 않는지 확인한다.
- 리트라이 진입점이 일반 웨이브 1인지 확인하되 초기화 범위는 별도로 검증한다.

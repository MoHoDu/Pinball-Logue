# flippers 기능 에이전트 규칙

## 책임

- 방향 입력으로 작동할 플리퍼를 즉시 선택한다.
- 선택된 플리퍼의 작동, 유지, 복귀와 재사용 가능 상태를 관리한다.
- 타격 시점과 접촉 조건에 따른 공 패링 결과를 물리 이벤트로 제공한다.

## 규칙

- 플리퍼 타이밍은 경로 선택을 실행하는 수단이며 자동 경로 결정을 대신하지 않는다.
- 선택 상태와 작동 상태를 분리하고 선택되지 않은 플리퍼의 우발 작동을 막는다.
- 회전 범위, 속도, 힘, 유지 시간과 복귀 시간은 설정값으로 노출한다.
- 프레임률에 따라 작동 결과가 크게 달라지지 않도록 물리 틱 기준으로 처리한다.
- 플리퍼는 점수, 웨이브 상태와 UI를 직접 변경하지 않는다.

## 필수 검증

- 모든 방향에서 올바른 플리퍼 하나가 선택되는지 확인한다.
- 이른·정확한·늦은 입력의 공 각도, 속도 또는 거리를 같은 조건에서 비교한다.
- 연타 시 관통, 중복 힘 적용, 복귀 불능과 입력 잠김이 없는지 확인한다.
- 낮은 프레임률과 정상 프레임률에서 핵심 타격 결과를 비교한다.

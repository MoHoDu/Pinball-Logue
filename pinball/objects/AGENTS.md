# objects 기능 에이전트 규칙

## 책임

- 공과 충돌하는 범퍼, 페그, 벽과 드레인의 재사용 가능한 동작을 제공한다.
- 충돌 세기, 대상 식별자와 발생 시점을 중립 이벤트로 전달한다.
- 코인과 웨이브 아이템 센서는 수집 사실만 전달하고 자원·효과 정책은 계산하지 않는다.

## 규칙

- 오브젝트 동작과 스테이지별 위치·개수를 분리한다.
- 범퍼는 반동과 기본 점수 근거를 제공할 수 있지만 최종 스코어를 직접 누적하지 않는다.
- 드레인은 낙하 이벤트를 한 번만 만들고 스테이지 실패를 선언하지 않는다.
- 동일 물리 접촉에서 이벤트가 프레임마다 중복 발생하지 않도록 접촉 단위를 명확히 한다.
- 보스와 유물처럼 별도 기능이 소유하는 오브젝트를 이 디렉토리에 복제하지 않는다.

## 필수 검증

- 각 충돌에서 의도한 이벤트 횟수와 반동 방향을 확인한다.
- 드레인과 수집 센서가 공 또는 수집물 하나당 한 번만 결과를 내는지 확인한다.
- 고속 공의 관통, 경계 끼임과 충돌 이벤트 누락을 확인한다.
- 스코어, 코인과 웨이브 아이템 이벤트가 구분되는지 확인한다.

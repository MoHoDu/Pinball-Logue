# Pinball_Logue 표준 용어

이 문서는 기획, 코드, 노드, UI, Jira, QA와 발표 자료에서 사용하는 게임 용어의 로컬 기준이다. 사용자가 변경을 지시하면 이 문서를 갱신한 뒤 새 기준을 적용한다.

## 전체 진행 구조

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 메인 로비 | 현재 진행 상황과 스테이지 1·2·3을 확인하고 진입할 스테이지를 선택하는 화면. `월드맵`, `스테이지 선택 화면`과 혼용하지 않는다. |
| 런 | 메인 로비에서 시작해 스테이지 1→2→3을 진행하는 전체 플레이 단위. |
| 스테이지 | 일반 웨이브 3개와 보스 웨이브 1개로 구성되는 큰 진행 단위. `웨이브`와 혼용하지 않는다. |
| 웨이브 | 하나의 핀볼 보드에서 목표 스코어를 달성하거나 실패할 때까지 진행하는 개별 플레이 단위. |
| 일반 웨이브 | 범퍼, 콤보와 웨이브 아이템으로 목표 스코어를 달성하는 웨이브. 현재 프로토타입에서는 일반 몬스터를 사용하지 않는다. |
| 보스 웨이브 | 스테이지 마지막에 등장해 앞선 웨이브의 빌드와 플리퍼 숙련을 검증하는 웨이브. 스테이지당 1개를 기본으로 한다. |
| 웨이브 클리어 | 일반 웨이브의 목표 스코어를 달성한 상태. 이후 보상 선택 화면으로 이동한다. |
| 스테이지 클리어 | 보스 웨이브를 완료한 상태. 다음 스테이지 개방 또는 메인 로비 복귀로 연결한다. |
| 스테이지 실패 | 해당 스테이지에서 허용된 생존 기회를 모두 소진한 상태. 연출에서 `사망`을 쓸 수 있지만 시스템 용어는 이 표현을 사용한다. |
| 리트라이 | 스테이지 실패 후 현재 스테이지의 일반 웨이브 1부터 다시 시작하는 것. 이전 스테이지까지 되돌아가지 않는다. |

## 핀볼 플레이와 조작

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 핀볼 보드 | 공, 플리퍼, 범퍼, 유물과 웨이브 아이템이 배치되는 실제 플레이 공간. |
| 공 | 플레이어가 발사하고 플리퍼로 개입하는 핵심 플레이 도구. 한 번에 보드에 1개만 존재한다. |
| 발사 | 공 하나가 보드에 투입되어 낙하하거나 결과가 확정될 때까지의 한 차례. `한 공`, `한 번의 플레이`와 혼용하지 않는다. |
| 플리퍼 | 다각형 보드 외곽의 여러 방향에 고정되어 공을 반사하거나 목표 경로로 보내는 조작 장치. |
| 플리퍼 선택 | 현재 작동시킬 플리퍼를 방향 입력 한 번으로 즉시 지정하는 행동. |
| 플리퍼 작동 | 선택된 플리퍼를 움직여 공을 타격하는 행동. 경로 선택을 실행하는 타이밍 입력이다. |
| 공 패링 | 적절한 타이밍에 플리퍼로 공을 강하게 타격해 속도, 거리와 타격감을 강화하는 행위. |
| 경로 선택 | 점수, 위험, 유물과 보스 위치를 고려해 공을 보낼 목표 경로를 결정하는 한 발사의 핵심 판단. |
| 순간 개입 | 자동으로 튀는 공에 플리퍼 선택과 타이밍으로 결정적인 영향을 주는 행동. 모든 이동을 직접 조종하지 않는다. |
| 예측 불가능성 | 물리 충돌로 인해 계획과 실제 결과에 차이가 생기는 성질. 불공정보다 기대 이상의 결과를 만드는 데 사용한다. |
| 운적인 성공 | 예상보다 좋은 충돌 연쇄와 스코어가 발생하는 경험. 아무 판단 없는 자동 승리는 만들지 않는다. |
| 낙하 | 공이 보드의 방어 영역을 벗어나 해당 발사가 끝난 상태. 한 번의 낙하와 스테이지 실패를 구분한다. |
| 공 라이프 | 공이 낙하했을 때 소모되는 재시도 자원. 수량과 적용 단위는 아직 미확정이다. |

## 점수와 충돌

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 스코어 | 범퍼, 보스, 웨이브 아이템과 오브젝트 충돌로 획득하는 일반 웨이브의 핵심 진행 수치. |
| 목표 스코어 | 웨이브를 클리어하기 위해 필요한 스코어. 플레이 중 계속 확인할 수 있어야 한다. |
| 범퍼 | 공의 방향을 바꾸고 스코어를 제공하는 일반 웨이브의 기본 오브젝트. |
| 충돌 | 공이 범퍼, 보스, 유물 또는 기타 오브젝트와 접촉해 반응을 발생시키는 것. 점수, 소리와 반동을 즉시 표시한다. |
| 콤보 시간 | 연속 충돌을 하나의 콤보로 인정하는 제한 시간. 시간 안에 추가 충돌하면 콤보를 유지한다. |
| 콤보 | 콤보 시간 안에 범퍼나 목표물을 연속 타격한 상태. 연속 횟수나 조건에 따라 스코어 배수가 증가한다. |
| 스코어 배수 | 획득 스코어에 곱해지는 배율. 콤보와 웨이브 아이템 등으로 증가한다. |
| 스코어 폭발 | 여러 충돌과 배수 효과가 겹쳐 짧은 시간에 스코어가 크게 상승하는 경험. 최우선 핵심 재미다. |
| 웨이브 아이템 | 웨이브 도중 보드에서 직접 획득하며 스코어 배수 상승 등 즉시 또는 일시 효과를 주는 이로운 아이템. |
| 보스 타격 | 공이 보스에게 직접 충돌해 HP 또는 기믹 진행도를 감소시키는 것. 범퍼보다 강한 피드백을 제공한다. |

## 보상과 자원

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 코인 | 일반 웨이브 보드에서 획득하고 보상 화면에서 소비하는 자원. 스코어와 별도의 수치와 표현을 사용한다. |
| 자원 관리 | 획득한 코인을 어떤 보상 카드에 사용할지 판단하는 과정. |
| 보상 선택 화면 | 일반 웨이브 클리어 후 랜덤 보상 카드 3개가 등장하는 화면. 화면 이름은 `보상 선택`을 유지한다. |
| 보상 카드 | 구매 가능한 유물, 패시브 효과와 가격 정보가 묶인 상품 단위. 웨이브 종료 후 3개가 무작위로 등장한다. |
| 보상 구매 | 보유 코인을 지불해 보상 카드를 획득하는 행동. 실제 행동은 `선택`보다 `구매`로 표현한다. |
| 카드 가격 | 보상 카드를 구매하는 데 필요한 코인의 양. 모든 카드에 명확히 표시한다. |
| 안정 보상 | 빌드의 기본 성능을 확실히 높이는 낮은 위험의 보상. 매 보상 화면에서 최소 2개를 살 수 있도록 경제를 설계한다. |
| 랜덤 보상 | 웨이브 종료 후 무작위로 제시되는 카드. 항상 같은 조합이 반복되지 않도록 한다. |

모든 보상 카드는 다음 정보를 한 화면에 표시한다.

1. 배치 가능한 유물의 종류와 수
2. 공에 적용되는 패시브 효과
3. 카드 구매 가격

## 성장과 빌드

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 유물 | 보상 카드로 획득하고 보드에 배치해 공의 경로와 충돌 결과를 바꾸는 콘텐츠. 위치가 성능과 결과에 영향을 줘야 한다. |
| 유물 배치 | 획득한 유물을 핀볼 보드의 특정 위치에 설치하는 행동. 전투 전 또는 웨이브 사이에 진행한다. |
| 패시브 효과 | 별도 조작 없이 공에 지속적으로 적용되는 공격력, 스코어 또는 충돌 성능 강화 효과. |
| 공 성장 | 보상과 패시브를 통해 공이 강해지고 플레이 방식이 달라지는 경험. 시각, 수치와 플레이에서 체감돼야 한다. |
| 보드 빌딩 | 유물을 배치해 고득점, 보스 공격 또는 안전 경로를 설계하는 전략. `덱 빌딩` 대신 쓰는 공식 용어다. |
| 빌드 | 획득한 유물, 유물 위치, 패시브 효과와 공 특성이 결합된 플레이 전략. 최소 해당 스테이지 보스까지 누적한다. |
| 수집 요소 | 여러 유물을 획득하고 조합하는 장기적 동기. 수집 자체보다 실제 배치와 사용 경험을 우선한다. |
| 디펜스 요소 | 유물 배치로 공 손실을 막거나 안전 경로를 만드는 보드 빌딩 요소. |
| 안전 경로 | 공의 생존과 안정적인 스코어 누적에 유리한 낮은 위험의 경로. |
| 고득점 경로 | 위험이 높지만 높은 배수와 스코어를 기대할 수 있는 경로. |
| 보스 공격 경로 | 공을 보스에게 반복적으로 보내는 보드상의 흐름. 보스 위치와 유물 배치를 함께 고려한다. |

## 적과 보스

| 용어 | 정의 및 사용 기준 |
| --- | --- |
| 보스 | 스테이지 마지막에 등장해 공 성장, 플리퍼 숙련과 보드 빌드를 검증하는 현재 프로토타입의 유일한 직접 공격 대상. |
| 보스 HP·진행도 | 공 타격으로 감소시키는 보스의 내부 수치 또는 기믹 진행값. 구체적인 방식은 보스별로 결정한다. |
| 보스 기믹 | 보스마다 존재하는 대표 공격 또는 대응 규칙. 프로토타입에서는 하나를 중심으로 제작한다. |
| 일반 몬스터 | 일반 웨이브에서 처치하는 고정형 또는 이동형 적. 현재 프로토타입 범위에서는 제외한다. |
| 오브젝트 | 범퍼, 보스, 유물 등 공과 충돌해 반응하는 요소를 포괄하는 상위 용어. |

## 혼용 금지 표현

| 사용하지 않을 표현 | 표준 표현 |
| --- | --- |
| 사각형 플리퍼 보드 | 다각형 외곽 플리퍼 보드 |
| 웨이브 일반스테이지 1 | 일반 웨이브 1 |
| 스코어 달성 시 다음 스테이지 | 웨이브 클리어 후 보상 구매 및 다음 웨이브 |
| 3개 보상 중 1개 선택 | 랜덤 보상 카드 3개 중 코인으로 구매 |
| 덱 빌딩 | 보드 빌딩 / 유물 배치 빌드 |
| 영구적 죽음 | 스테이지 리트라이 |
| 플레이어 사망 | 스테이지 실패 |
| 플리퍼 타이밍이 핵심 판단 | 경로 선택이 핵심 판단, 타이밍은 실행 수단 |
| 일반 몬스터 | 현재 제외 콘텐츠 |
| 아이템 | 웨이브 아이템 / 유물 / 보상 카드 |
| 보상 선택 | 화면: 보상 선택 / 행동: 보상 구매 |
| 다음 판 | 다음 웨이브 / 다음 스테이지 / 리트라이 |

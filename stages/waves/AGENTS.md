# stage waves 에이전트 규칙

## 책임

- 일반 웨이브 하나가 사용할 보드, 목표와 콘텐츠 설정을 조합한다.
- 범퍼 구성, 코인 및 웨이브 아이템 출현 설정을 참조한다.
- 스테이지 안의 일반 웨이브 1·2·3이 점진적으로 다른 경로 판단을 제공하게 구성한다.

## 규칙

- 일반 웨이브에는 현재 프로토타입 범위에서 일반 몬스터를 사용하지 않는다.
- 웨이브 데이터는 시작·클리어·실패 상태를 관리하거나 보상 화면을 열지 않는다.
- 대부분의 주요 충돌이 목표 스코어에 기여하도록 구성한다.
- 코인 획득은 스코어와 별도 콘텐츠로 설정하고 결과 이벤트도 구분한다.
- 특정 보드, 목표와 아이템의 참조가 유효한지 실행 전에 검증한다.

## 필수 검증

- 각 일반 웨이브의 모든 참조가 존재하고 호환되는지 확인한다.
- 목표 진행에 기여하지 않는 충돌의 비율과 진행 정체 구간을 측정한다.
- 웨이브별 목표 시간, 평균 발사 수와 실패 빈도를 같은 조건에서 비교한다.
- 일반 몬스터 데이터나 동작이 포함되지 않았는지 확인한다.

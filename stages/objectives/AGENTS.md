# stage objectives 에이전트 규칙

## 책임

- 일반 웨이브에 적용할 목표 스코어 설정을 제공한다.
- 보스 HP 또는 기믹 진행도처럼 보스별 완료 조건을 데이터로 제공한다.
- 목표 식별자, 단위와 비교 기준을 명확하게 정의한다.

## 규칙

- 일반 웨이브 목표는 스코어 달성을 기본으로 하고 특정 작은 표적 하나에 종속시키지 않는다.
- 목표 데이터는 조건을 제공할 뿐 스코어, 보스 피해나 진행 상태를 직접 변경하지 않는다.
- 목표 달성 계산 결과는 한 번만 발생해야 하며 상태 전환은 `game_flow/`가 담당한다.
- 목표값은 웨이브 설정에서 참조하고 코드에 스테이지별 숫자를 하드코딩하지 않는다.
- 공 라이프와 스테이지 실패 조건은 `D-01`, `D-02` 결정 전까지 목표에 포함하지 않는다.

## 필수 검증

- 0, 음수, 누락되거나 지원하지 않는 목표 설정을 로드 전에 거부하는지 확인한다.
- 목표값과 실제 계산 단위가 일치하는지 확인한다.
- 목표를 한 번에 초과해도 달성 결과가 한 번만 발생하는지 확인한다.
- 목표 변경 전후의 평균 발사 수, 소요 시간과 스코어 분포를 같은 조건에서 비교한다.

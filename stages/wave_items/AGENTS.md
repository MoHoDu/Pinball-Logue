# wave items 에이전트 규칙

## 책임

- 웨이브 아이템의 식별자, 출현 조건, 지속 시간과 효과 설정을 제공한다.
- 획득 시 즉시 또는 일시적으로 적용할 중립 효과 명령을 제공한다.
- 스코어 배수 등 이로운 효과의 시작과 종료를 진단 가능한 이벤트로 제공한다.

## 규칙

- 웨이브 아이템은 보드에서 직접 획득하며 코인, 유물과 보상 카드로 분류하지 않는다.
- 아이템 데이터는 물리 센서, 스코어 계산, UI와 이펙트 구현을 직접 소유하지 않는다.
- 획득 하나는 한 번만 적용하고 지속 효과는 종료 시점을 명시한다.
- 현재 프로토타입에서는 예고 없는 불이익보다 예상 이상의 충돌 연쇄와 보너스 결과를 만든다.
- 외부 기능에 적용하는 효과는 승인된 공개 계약을 사용한다.

## 필수 검증

- 중복 접촉에도 아이템 하나가 한 번만 획득되는지 확인한다.
- 즉시 효과의 값과 지속 효과의 시작·종료 시간이 설정과 일치하는지 확인한다.
- 아이템 획득 전후의 스코어, 배수 또는 충돌 결과 차이를 같은 조건에서 비교한다.
- 코인, 유물과 보상 카드의 식별자 및 이벤트와 혼동되지 않는지 확인한다.

# shared ids 에이전트 규칙

- 표시 이름과 식별자를 분리하고 식별자는 생성 뒤 변경하지 않는다.
- 서로 다른 도메인의 식별자를 같은 문자열 타입처럼 혼용하지 않는다.
- 빈 값, 잘못된 형식과 도메인 불일치를 생성·역직렬화 시 거부한다.
- 생성 방식은 Web과 로컬 실행에서 충돌 가능성을 통제하고 테스트에서 고정값 주입을 허용한다.
- 중복 생성, 직렬화 왕복과 오래된 참조의 오류 처리를 검증한다.

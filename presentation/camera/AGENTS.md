# presentation camera 에이전트 규칙

- 공, 플리퍼 개입 지점과 보스 위험 범위가 동시에 읽히는 프레이밍을 우선한다.
- 추적 속도, 줌, 흔들림 강도·시간과 감쇠는 설정값으로 노출한다.
- 화면 흔들림은 충돌 결과를 표현하지만 공 위치를 잃게 하거나 물리 결과를 바꾸지 않는다.
- 중첩 흔들림의 상한과 사용자 감소 옵션을 제공할 수 있는 구조를 유지한다.
- 최대 공 속도, 보스 공격과 작은 Web 뷰포트에서 공이 화면 밖으로 사라지는 시간을 측정한다.

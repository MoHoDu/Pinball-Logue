# presentation feedback 에이전트 규칙

- 범퍼, 공 패링, 콤보 임계값과 보스 직접 타격에 구분되는 피드백 프리셋을 제공한다.
- 히트 스톱은 표현 계층의 시간 효과로 제한하고 물리 상태나 입력 결과를 손상시키지 않는다.
- 강도, 지속 시간과 동시 재생 상한을 두고 짧은 시간의 반복 타격에서도 공을 추적 가능하게 한다.
- 보스 강한 공격은 준비→위험 범위→발동→결과 단계를 시청각으로 구분한다.
- 피드백을 끄거나 낮춰도 충돌·위험·결과를 이해할 대체 표시가 남는지 검증한다.
